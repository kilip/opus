# Opus

Opus is an **open source, self-hosted, autonomous AI assistant** designed to operate 24/7 on behalf of individuals and teams. It unifies a structured knowledge base, a programmable workflow engine, and a proactive AI agent layer into a single platform.

## 🚀 Core Principles

- **Local-first**: Your data stays on your infrastructure.
- **Privacy by default**: No telemetry, no vendor lock-in, no cloud dependency.
- **Zero mandatory cost**: Fully functional with free and open source LLM providers.

## ✨ Features

- **Autonomous Agent**: Works 24/7 without needing constant manual triggers.
- **Unified Knowledge**: Connects your notes, tasks, and context.
- **Programmable Workflows**: Powerful engine to automate your routine.
